    Intro to Python

    Gain foundational knowledge of Python programming, including syntax, data types, control structures, functions, and simple projects.

    Key Topics Covered

    This beginner-friendly course introduces students to Python, one of the most popular and versatile programming languages. With hands-on coding examples and real-world applications, learners will grasp the fundamentals of programming logic and syntax.

    • Python Basics
      • Installing and running Python
      • Syntax, variables, and data types
      • Operators and expressions
    • Control Flow
      • Conditional statements (if, elif, else)
      • Loops: for and while
      • Logical operations
    • Functions and Modules
      • Defining and calling functions
      • Parameters and return values
      • Importing standard modules
    • Working with Data
      • Lists, tuples, and dictionaries
      • Reading and writing files
      • Basic string manipulation

    Learning Outcomes

    • Understand core Python syntax and structure
    • Write simple programs and solve basic problems
    • Develop logical thinking through code
